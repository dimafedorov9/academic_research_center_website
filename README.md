# Научная кафедра — МАОУ «Гимназия «Краснообская»

Веб-сайт образовательного кластера «Школа-НИИ»

## Описание проекта

Сайт научной кафедры гимназии, созданной при поддержке СибНИИЗБиЦ СФНЦА РАН. Проект направлен на интеграцию научно-исследовательской деятельности в образовательный процесс школы.

## Структура сайта

- **Главная** (`index.html`) — о проекте, статистика, направления работы
- **О кафедре** (`about.html`) — концепция, цели, методология
- **Команда** (`team.html`) — учёные РАН и педагоги-кураторы
- **Направления** (`directions.html`) — научные области: анализ данных, биология, почвоведение, ГИС
- **Проекты** (`projects.html`) — исследовательские работы учащихся
- **Курсы** (`courses.html`) — образовательные программы (курс «Анализ данных», практикумы)
- **Ученикам** (`for-students.html`) — как стать исследователем
- **Учителям** (`for-teachers.html`) — услуги кафедры для педагогов
- **Новости** (`news.html`) — события и достижения
- **Контакты** (`contacts.html`) — способы связи

## Деплой на GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите все файлы из этой папки
3. В настройках репозитория включите GitHub Pages (Settings → Pages)
4. Выберите ветку `main` и корневую папку `/`
5. Сайт будет доступен по адресу: `https://username.github.io/repository-name/`

## Локальный запуск

Просто откройте `index.html` в браузере. Все ресурсы подключены через CDN, дополнительная установка не требуется.

## Обновление контента

Для обновления информации на сайте редактируйте соответствующие HTML-файлы. Основные места для изменений:

- **Статистика** — `index.html`, блок `.stats-grid`
- **Новости** — `news.html`, добавляйте новые `.team-card`
- **Проекты** — `projects.html`, блок с проектами
- **Команда** — `team.html`, карточки специалистов
